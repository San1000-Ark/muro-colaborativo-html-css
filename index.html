Glosario Técnico

Commit
Registro de cambios realizado en el proyecto.
Repositorio
Carpeta donde se almacena el código fuente y su historial.
Branch
Línea alternativa de desarrollo dentro de un repositorio.

Preguntas Frecuentes

¿Qué es Git?

Git es un sistema de control de versiones distribuido para manejar proyectos.

¿Para qué sirve GitHub?

GitHub permite alojar proyectos Git en la nube y colaborar con otros desarrolladores.
